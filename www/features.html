Gravity Features

These are some of the features implemented in Gravity. Feel free to discuss new features in the mailing list.

  • Serves components and configuration: Gravity, as a container, serves components and static configuration data in the form of lists ad maps.
  • Component and configuration injection: Components can be injected with other dependent components and static configuration data.
  • Constructor and method injection: Dependencies can be injected into components via constructors or any method of the component.
