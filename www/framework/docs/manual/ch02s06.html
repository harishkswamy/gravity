Modules

Modules are simply a collection of components. The manner in which components are specified within the modules is entirely plugin specific.

Example 2.2. Module Specification File (BeanShell)

import components.pkg;

// Register a component implementation
componentImpl
(
    // Component being registered
    ComponentA.class,

    // Register dependencies via constructor injection
    constructorArgs( "firstArg", 23.1 ),

    // Register life cycle methods
    callbacks
    (
        // Register dependencies via method injection
        injection("setComponentB", componentInst( ComponentB.class )),
        injection("setComponentC", componentInst( ComponentC.class )),

        // Register startup methods
        startup("initialize")
    )
);
