Gravity is a runtime container that will gather, configure, manage and serve software components.

  • Gravity will gather the components by sweeping the classpath looking for plugins - a group of related software components. The plugins are identified by the plugin manifest file, gravity-plugin.properties, placed under the META-INF directory in the classpath.

  • Gravity will configure the components as instructed by the plugins. Gravity is capable of configuring components via constructor-, method- and combo-injection techniques.

  • Gravity will manage the components via a predefined set of life cycle phases: instantiation, injection, start-up and shutdown.

  • Gravity will serve the gathered components, fully configured, when requested.
